# Aiper Pro Store

Kvalitnejší základ postavený bez Tailwindu a bez zbytočných závislostí.

## Prečo takto
- Next.js je pripnutý na `16.0.10`
- React je pripnutý na `19.2.1`
- bez Tailwind/PostCSS konfliktov
- čisté komponenty
- kvalitný custom vizuál cez CSS
- pripravené na Vercel

## Spustenie
```bash
npm install
npm run build
npm run dev
```

## Deploy na Vercel
1. Nahraj obsah priečinka `aiper-pro-store` do GitHub repozitára.
2. Skontroluj, že v GitHube je `package.json` s `"next": "16.0.10"`.
3. Vo Verceli daj Redeploy without cache.
